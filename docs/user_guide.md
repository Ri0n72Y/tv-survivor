# 用户操作指南

## 当前目标

- 在阵列中探索房间、获取金币和奖励，逐步强化武器与被动。
- 清理 Boss 房后，本局胜利。
- 按 `R` 可以随时重新开始本局。

## 阵列探索

- 阵列界面底部会显示当前地图种子；右侧输入框可填写下次使用的地图种子，留空点击“重新开始”会自动生成新种子。
- 输入框中的种子只会在点击“重新开始”后生效，不会立即刷新当前地图。
- 地图默认由当前种子生成；相同种子在相同版本和相同行为顺序下会得到可复现的阵列内容。
- 使用 `WASD` 或方向键在阵列中移动。
- 只能进入已经揭示、相邻、非障碍的格子。
- 当前所在格会显示高亮和“你”角标。
- 主角开局默认拥有基础弹 Lv.1。
- 进入未清理的任务、搜索、精英或 Boss 格会开始一场战斗。
- 战斗成功后返回阵列；战斗失败会回到上一个格子。
- 同步率较高地完成战斗时，会额外揭示周围区域。

## 格子类型

- 起点：本局起始位置。
- 任务：普通幸存者战斗。
- 搜索：包含战斗宝箱、祭坛挑战和撤离读条。
- 精英：封闭竞技场，击杀精英后选择奖励。
- Boss：封闭竞技场，击杀 Boss 后本局胜利。
- 宝箱：阵列宝箱，需要金币开启。
- 障碍：不能进入。

## 战斗基础

- 使用 `WASD` 或方向键移动角色。
- 敌人接触玩家会造成同步率损失；玩家受击后会出现受击特效，并获得短暂无敌时间。
- 普通任务房和搜索房使用同步圈：尽量留在信号范围内，避免边缘衰减和同步断开。
- 精英房和 Boss 房是封闭圆形竞技场：保留同步率生命值，敌人接触仍会造成同步率损失，但没有同步圈的边缘衰减和断连强制返回。
- 普通任务持续 30 秒；时间结束后进入 3 秒撤离倒计时，撤离期间不再刷新敌人。
- 战斗难度会随时间每 10 秒提升一次；玩家武器与被动总等级只会按对数平缓影响刷怪量和高等级怪比例。
- 高等级怪沿用同一敌人类型，用更深的颜色表示，拥有更高血量并掉落更多金币，但不会因为等级提高而额外加速。
- 敌人死亡会掉落绿色金币收集物，靠近后自动吸附并拾取。
- `Esc` 仅作为调试失败出口，不是正式撤离或失败操作。

## Search 房

- 搜索房内会每 10 秒刷新一个战斗宝箱。
- 战斗宝箱刷新在中心附近，靠近后自动尝试打开。
- 普通战斗宝箱消耗 30 金币；价格在宝箱刷新时固定，金币不足时不会打开。
- 打开宝箱后从三项奖励中选择一项。
- 三个战斗宝箱都刷新后，祭坛在中心出现。
- 在祭坛圈内停留 3 秒会激活挑战；离开范围后读条不会立刻清零，而是按相同速度缓慢下降。
- 祭坛挑战持续 20 秒，激活时会刷新精英，挑战期间敌人刷新强度提高。
- 坚持 20 秒或击杀精英后，撤离点开放。
- 回到祭坛圈内停留 3 秒即可返回阵列；离开范围后撤离读条同样会缓慢下降。

## 精英房与 Boss 房

- 精英房没有同步圈，圆形房间边界会阻挡玩家离开。
- 精英房计时归零后会提高难度。
- 击杀精英会吸收场上掉落，并弹出奖励选择。
- Boss 房不限时，目标是击杀 Boss；敌人接触仍会造成同步率损失。
- 击杀 Boss 后会吸收场上掉落，并在阵列界面显示本局胜利。

## 奖励与构筑

- 奖励包括新武器、武器升级、新被动和被动升级。
- 当前武器：基础弹、光环、固定形状、射线。
- 当前被动：移动速度、全武器伤害、冷却缩短、金币吸附、同步强化、金币收益。
- 打开奖励面板时战斗暂停。
- 使用 `WASD` / 方向键选择奖励卡，按空格或回车确认。

## 金币与宝箱

- 金币是当前局内消费资源。
- 金币来自敌人掉落和收益倍率加成。
- 阵列宝箱和战斗宝箱都会消耗金币。
- 阵列宝箱更多，价格会在格子被揭示时固定，并随当前总等级对应的难度等级增加；战斗宝箱价格在刷新时固定。
- 宝箱类型和奖励由当前局的确定性随机系统决定；同一 run seed 在相同行为顺序下会得到相同结果。
- 重新开始后金币、地图进度、武器和被动都会重置。

## 确定性随机

- 当前项目使用 run seed 驱动整局随机。
- 地图、刷怪、宝箱类型、宝箱奖励、武器奖励、被动奖励、商店刷新等使用彼此独立的命名随机流。
- 一个系统多消耗随机数不会污染其他系统。例如，战斗刷怪变化不会改变商店刷新，地图生成变化不会改变宝箱奖励。
- 非玩法表现随机，例如动画、粒子或音效抖动，不应消耗 gameplay RNG。
- RNG 接入和存档约定见 `docs/rng.md`。
