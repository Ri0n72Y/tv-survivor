# 用户操作指南

## 阵列探索

- 使用 `WASD` 或方向键在阵列中移动。
- 只能进入已经揭示、相邻、非障碍的格子。
- 鼠标点击不用于阵列移动。
- 主角默认拥有基础弹 Lv.1。
- 进入宝箱格会消耗金币，并提升一个未满级武器。
- 进入未完成的任务、精英或 Boss 格会开始一场战斗。

## 战斗任务

- 使用 `WASD` 或方向键移动角色。
- 普通任务房使用同步圈：尽量留在信号范围内，避免边缘衰减和同步断开。
- Search 房使用同步圈：房内会每 10 秒刷新一个战斗宝箱，三个宝箱刷新后祭坛出现在中心。
- 精英房和 Boss 房是封闭竞技场：没有同步圈，边缘只作为墙，不能向圈外移动。
- 普通任务持续 30 秒；时间结束后进入 3 秒撤离倒计时，撤离期间不再刷新敌人。
- 精英房计时归零后会提高难度；击杀精英会全屏吸收掉落，并触发一次奖励宝箱后返回阵列。
- Boss 房不限时；清理 Boss 后会全屏吸收掉落并获得本局胜利。
- 战斗难度会随时间每 10 秒提升一次，并额外根据当前所有武器等级总和提高。
- 敌人死亡会掉落绿色收集物，靠近后自动拾取并增加金币。
- 按 `R` 可重新开始本局。
- `Esc` 仅作为调试失败出口使用，不是正式撤离或失败操作。

## Search 房

- 战斗宝箱会刷新在中心附近；靠近后自动尝试打开。
- 普通宝箱消耗 30 金币，宝箱上会标注所需金币。
- 金币不足时不会打开宝箱，并会显示提示。
- 打开宝箱后从三项奖励中选择一项；当前奖励包括武器升级、新武器、同步率恢复、金币返还。
- 打开奖励面板时战斗完全暂停；使用 `WASD` / 方向键选择三张卡，按空格或回车确认。
- 三个宝箱都刷新后，祭坛在地图中心出现。
- 在祭坛圈内停留 3 秒激活挑战。
- 祭坛挑战持续 20 秒，激活时会刷新精英，挑战期间敌人刷新强度提高。
- 坚持 20 秒或击杀精英后，可以回到祭坛圈内停留 3 秒成功返回阵列。

## 金币与宝箱

- 金币是当前实际消费资源。
- 宝箱开启会消耗金币，并在四种武器中选择升级。
- 重新开始后金币清零。
