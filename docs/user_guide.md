# 用户操作指南

## 阵列探索

- 使用 `WASD` 或方向键在阵列中移动。
- 只能进入已经揭示、相邻、非障碍的格子。
- 鼠标点击不用于阵列移动。
- 主角默认拥有基础弹 Lv.1。
- 进入宝箱格会消耗金币，并提升一个未满级武器。
- 进入未完成的任务、精英或 Boss 格会开始一场战斗。

## 战斗任务

- 使用 `WASD` 或方向键移动角色。
- 普通任务房使用同步圈：尽量留在信号范围内，避免边缘衰减和同步断开。
- 精英房和 Boss 房是封闭竞技场：没有同步圈，边缘只作为墙。
- 普通任务持续 30 秒；时间结束后进入 3 秒撤离倒计时，撤离期间不再刷新敌人。
- 精英房和 Boss 房需要在时间结束前击杀目标；击杀精英会自动收集掉落并撤离，清理 Boss 后本局胜利。
- 战斗难度会随时间每 10 秒提升一次，并额外根据当前所有武器等级总和提高。
- 敌人死亡会掉落绿色收集物，靠近后自动拾取并增加金币。
- 按 `R` 可重新开始本局。
- `Esc` 仅作为调试失败出口使用，不是正式撤离或失败操作。

## 金币与宝箱

- 金币是当前实际消费资源。
- 宝箱开启会消耗金币，并在四种武器中选择升级。
- 重新开始后金币清零。
