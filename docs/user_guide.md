# 用户操作指南

## 阵列探索

- 使用 `WASD` 或方向键在阵列中移动。
- 只能进入已经揭示、相邻、非障碍的格子。
- 鼠标点击不再用于阵列移动。
- 进入宝箱格会提升一个未满级武器。
- 进入未完成的任务格会开始一场战斗。

## 战斗任务

- 使用 `WASD` 或方向键移动角色。
- 尽量留在信号范围内，避免同步率归零。
- 每场任务持续 30 秒；任务时间结束后进入 3 秒撤离倒计时，撤离期间不再刷新敌人。
- 战斗难度会随时间每 10 秒提升一次，并额外根据当前所有武器等级总和提高。
- 击杀普通敌人和精英敌人都会获得积分。
- 敌人死亡会掉落绿色收集物，靠近后自动拾取并增加积分。
- 击杀精英会触发一次全场收集，并立即进入撤离倒计时。
- 按 `R` 可重新开始本局。

## 分数

- 阵列界面和战斗 HUD 都会显示当前总分。
- 总分来自击杀奖励和收集物奖励。
- 重新开始后总分清零。
